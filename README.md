# SupernovaMultipoleStudy
This is the repository for supernova simulation research.  The goal is to run simulations of supernovae that recreate the quadrapole and octapole moments of real supernovae observations.

config.py is the file containing various global variables for the analysis.

This is analysis code for the output of Sprout simulations of supernovae which come in the form of h5 files.  It was originally a jupyter notebook but is being transferred to python files
for better organization.
